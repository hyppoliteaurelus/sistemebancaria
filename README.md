# Conta Bancária Virtual

Este projeto é uma aplicação Java para controlar uma conta bancária virtual, onde é possível realizar operações como consultar saldo, receber e transferir valores.

        ## Funcionalidades

- Consultar saldo
- Receber valor
- Transferir valor
- Encerrar aplicação
